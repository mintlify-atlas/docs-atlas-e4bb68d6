---
title: "zoom_phone_user_calling_plans — Assign calling plans"
description: "Assign one or more calling plans to a Zoom Phone user, controlling the call types and regions available to them using zoom_phone_user_calling_plans."
---

Calling plans determine what kinds of calls a Zoom Phone user can make — for example, metered domestic calls, unlimited US/Canada calls, or international toll-free numbers. The `zoom_phone_user_calling_plans` resource manages the set of calling plans assigned to a user. The entire set is managed as a unit: applying a new plan list replaces any previously assigned plans for that user.

## Required API permissions

This resource requires the following OAuth scopes on the server-to-server app credential used by the provider:

- `phone:read:user:admin`
- `phone:write:calling_plan:admin`
- `phone:delete:users_calling_plan:admin`

## Example usage

```hcl
resource "zoom_phone_user" "example" {
  user_id          = "AAggbuS-Q6aXXcsv2wnug"
  extension_number = 101
}

resource "zoom_phone_user_calling_plans" "example" {
  user_id = zoom_phone_user.example.user_id
  calling_plans = [
    {
      type = 207
    },
  ]
}
```

## Import

Import existing calling plan assignments using the Zoom user ID:

```bash
# ${user_id}
terraform import zoom_phone_user_calling_plans.example AAggbuS-Q6aXXcsv2wnug
```
