---
title: "zoom_phone_auto_receptionist data source"
description: "Read configuration details of a Zoom Phone auto receptionist by ID, returning its name, extension, timezone, audio language, and associated site."
---

Use the `zoom_phone_auto_receptionist` data source to retrieve configuration details for a specific auto receptionist. Auto receptionists answer inbound calls with a personalized recording and route them to a phone user, call queue, common area, voicemail, or an IVR system. Reference this data source when you need the auto receptionist's extension number, site, or timezone in other resources or outputs.

## API permissions

The following OAuth scope is required to use this data source:

- `phone:read:auto_receptionist:admin`

## Example usage

```hcl
data "zoom_phone_auto_receptionist" "example" {
  id = "t6wyhAZRQXXX_Rv3jj3XXX"
}

output "example" {
  value = data.zoom_phone_auto_receptionist.example
}
```

## Schema

### Required arguments

<ParamField path="id" type="string" required>
  The unique identifier of the auto receptionist.
</ParamField>

### Returned attributes

<ResponseField name="name" type="string">
  The display name of the auto receptionist.
</ResponseField>

<ResponseField name="extension_number" type="number">
  The extension number assigned to the auto receptionist.
</ResponseField>

<ResponseField name="extension_id" type="string">
  The extension ID of the auto receptionist.
</ResponseField>

<ResponseField name="timezone" type="string">
  The timezone of the auto receptionist. See the [Zoom timezone abbreviation list](https://marketplace.zoom.us/docs/api-reference/other-references/abbreviation-lists#timezones) for valid values.
</ResponseField>

<ResponseField name="audio_prompt_language" type="string">
  The language used for all default audio prompts played by this auto receptionist.
</ResponseField>

<ResponseField name="cost_center" type="string">
  The cost center name associated with the auto receptionist.
</ResponseField>

<ResponseField name="department" type="string">
  The department name associated with the auto receptionist.
</ResponseField>

<ResponseField name="site" type="object">
  The site to which this auto receptionist is assigned.

  <Expandable title="properties">
    <ResponseField name="id" type="string">
      The unique identifier of the site.
    </ResponseField>
    <ResponseField name="name" type="string">
      The display name of the site.
    </ResponseField>
  </Expandable>
</ResponseField>
