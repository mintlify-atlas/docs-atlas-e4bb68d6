Use the `zoom_phone_phone_numbers` data source to list phone numbers provisioned in your Zoom account. You can retrieve all numbers or narrow the results using the `filter` block — for example, to find only unassigned toll-free numbers on a specific site. This data source is useful for looking up a number's ID or assignee before referencing it in other resources.

## API permissions

The following OAuth scope is required to use this data source:

- `phone:read:list_numbers:admin`

## Example usage

```hcl
# Fetch all phone numbers in the account
data "zoom_phone_phone_numbers" "all" {}

# Fetch only unassigned numbers
data "zoom_phone_phone_numbers" "filtered" {
  filter = {
    type = "unassigned"
  }
}

output "example" {
  value = data.zoom_phone_phone_numbers.all
}

output "filtered" {
  value = data.zoom_phone_phone_numbers.filtered
}

# Look up a specific number by its E.164 value
output "search_phone_number" {
  value = try(
    data.zoom_phone_phone_numbers.all.phone_numbers[
      index(data.zoom_phone_phone_numbers.all.phone_numbers.*.number, "+1234567890")
    ],
    null
  )
}
```

## Schema

### Optional arguments

<ParamField path="filter" type="object">
  Narrow the list of returned phone numbers.

  <Expandable title="properties">
    <ParamField path="type" type="string">
      Filter by assignment status. Allowed values:
      - `assigned` — numbers assigned to a user, call queue, auto receptionist, or common area.
      - `unassigned` — numbers not assigned to anyone.
      - `all` — include both assigned and unassigned numbers.
      - `byoc` — return only Bring Your Own Carrier (BYOC) numbers.
    </ParamField>
    <ParamField path="extension_type" type="string">
      Filter assigned numbers by the type of assignee. Only valid when `type` is `assigned`. Allowed values: `user`, `callQueue`, `autoReceptionist`, `commonArea`, `emergencyNumberPool`, `companyLocation`, `meetingService`.
    </ParamField>
    <ParamField path="number_type" type="string">
      Filter by number type. Allowed values: `toll`, `tollfree`.
    </ParamField>
    <ParamField path="site_id" type="string">
      The unique identifier of the site to filter results by. Only valid when multiple sites are enabled.
    </ParamField>
    <ParamField path="pending_numbers" type="boolean">
      When `true`, includes pending numbers in the response.
    </ParamField>
  </Expandable>
</ParamField>
