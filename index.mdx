## Get started

<Steps>
  <Step title="Create a Zoom Server-to-Server OAuth app">
    In the [Zoom Marketplace](https://marketplace.zoom.us/user/build), create a Server-to-Server OAuth app and collect your Account ID, Client ID, and Client Secret.
  </Step>
  <Step title="Configure the provider">
    Add the provider to your Terraform configuration and supply your credentials either directly or via environment variables.

    ```hcl terraform.tf
    terraform {
      required_providers {
        zoom = {
          source  = "folio-sec/zoom"
          version = "~> 0.1"
        }
      }
    }

    provider "zoom" {
      account_id    = var.zoom_account_id
      client_id     = var.zoom_client_id
      client_secret = var.zoom_client_secret
    }
    ```
  </Step>
  <Step title="Define your resources">
    Declare your Zoom Phone resources in Terraform and run `terraform apply`.

    ```hcl main.tf
    resource "zoom_phone_call_queue" "support" {
      name             = "Support Queue"
      extension_number = "200"
    }
    ```
  </Step>
</Steps>

<Note>
  The provider requires a Zoom account with Zoom Phone enabled. All resources use admin-level API permissions — ensure your OAuth app has the required scopes before applying.
</Note>
